## 📌 Como utilizar

### 💻 Backend
```bash 
#Entre no diretório do projeto
$ cd projeto_tarefas
```
```bash 
#Dê run no backend
$ python manage.py runserver
```
### 🌎 Frontend
```bash 
#Entre no diretório do frontend
$ cd todo-main
```
```bash 
#Instale as dependências do projeto
$ npm install
```
```bash 
#Dê run no frontend
$ ng serve -o
```
